# libmy

**my_getnbr**
Transform char* to int.
***Exemple***
```
int nbr = my_getnbr("234");
```
The value of nbr is 234.

**my_gettxt**
Transform int to char*.
***Exemple***
```
char *txt = my_gettxt(234);
```
The value of txt is "234".

**my_print_array**
Print char**.
***Exemple***
```
char **array = malloc(...)
...
array[0] = "Hello";
array[1] = "World";

my_print_array(array);

```
**output**
Hello
World
